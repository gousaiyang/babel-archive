Utility library for the Python `bpc <https://github.com/pybpc/bpc>`_ backport compiler.

Currently, the three individual tools (`f2format <https://github.com/pybpc/f2format>`_,
`poseur <https://github.com/pybpc/poseur>`_, `walrus <https://github.com/pybpc/walrus>`_)
depend on this repo. The bpc compiler is a work in progress.
